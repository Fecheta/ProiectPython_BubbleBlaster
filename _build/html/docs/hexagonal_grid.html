hexagonal_grid module

This module contains a class that is representing the hexagonal grid of the game, it’s main job is to manage a 2D array of HexagonalTile objects and it’s methods are focussed on doing that.

Bases: object

This is the class responsible for main grid of the game, it generates the grid, modify it and process the content of it, it also contains sound for the game.
